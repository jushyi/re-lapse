<task type="auto">
  <name>Task 1: Create COLOR_REFERENCE.md quick reference documentation</name>
  <files>src/constants/COLOR_REFERENCE.md</files>
  <action>
Create a scannable quick reference document with:

1. **Color System Overview** - Brief explanation of the color constant system

2. **Background Colors**
   - `colors.background.primary` - Pure black (#000000) - All screen backgrounds
   - `colors.background.secondary` - Dark gray (#111111) - Cards, sheets
   - `colors.background.tertiary` - Darker gray (#2A2A2A) - Nested elements
   - `colors.background.white` - White (#FFFFFF) - Light backgrounds (rare)

3. **Text Colors**
   - `colors.text.primary` - White - Main text
   - `colors.text.secondary` - Gray - Muted text, labels
   - `colors.text.tertiary` - Darker gray - Very muted
   - `colors.text.inverse` - Black - Text on light backgrounds

4. **Icon Colors**
   - `colors.icon.primary` - White - Default icons
   - `colors.icon.secondary` - Gray - Muted icons
   - Note: Icons should NOT use brand purple

5. **Interactive/Accent Colors**
   - `colors.interactive.primary` - Purple - Buttons, active states, highlights
   - `colors.brand.purple` - Same purple for brand elements
   - Note: Purple is for shapes and highlights, not icons

6. **Status Colors**
   - `colors.status.ready` - Green - Success states
   - `colors.status.danger` - Red - Errors, delete actions
   - `colors.status.developing` - Red - Developing state

7. **Border Colors**
   - `colors.border.subtle` - Dark gray - Subtle dividers

8. **Overlay Colors**
   - `colors.overlay.dark` - Semi-transparent black

9. **Usage Examples**

   ```javascript
   import { colors } from '../constants/colors';

   // Screen background
   backgroundColor: colors.background.primary

   // Card background
   backgroundColor: colors.background.secondary

   // Primary text
   color: colors.text.primary

   // Icon color
   <Ionicons color={colors.icon.primary} />

   // Interactive button
   backgroundColor: colors.interactive.primary
   ```

10. **Rules** - Always use constants, never hardcode hex values - Icons use white/gray, NOT purple - Purple is for interactive elements and highlights only - All screens must have `colors.background.primary` background
