**Color mapping reference:**

- '#000000' → colors.background.primary
- '#2A2A2A' → colors.background.tertiary (or colors.background.card for subtle cards)
- '#1A1A1A' → colors.background.secondary
- '#FFFFFF' → colors.text.primary or colors.icon.primary
- '#888888' → colors.text.secondary
- '#666666' → colors.text.tertiary
- '#FF3B30' → colors.status.danger
- '#333333' → colors.border.subtle
