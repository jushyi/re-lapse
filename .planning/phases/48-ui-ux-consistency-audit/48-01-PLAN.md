**Established patterns (from Phase 16 Color Constants):**

- colors.background.primary for all screen backgrounds
- colors.background.secondary for cards/sheets
- colors.text.primary/secondary/tertiary for text hierarchy
- colors.icon.\* for icons (NOT purple)
- colors.interactive.primary for buttons/active elements
- spacing.\* for all padding/margin values
- typography.fontFamily.\* for all font families
- typography.size.\* for all font sizes
- layout.borderRadius.\* for all border radius values
- layout.dimensions.\* for standard dimensions (inputHeight, buttonMinHeight, avatar sizes)

**Audit checklist per file:**

1. No hardcoded hex/rgb/rgba color values — use colors.\*
2. No hardcoded numeric padding/margin — use spacing.\*
3. No hardcoded font families — use typography.fontFamily.\*
4. No hardcoded font sizes — use typography.size.\*
5. No hardcoded border radius — use layout.borderRadius.\*
6. No hardcoded dimensions for standard elements — use layout.dimensions.\*
7. Consistent component patterns (buttons, inputs, headers)
