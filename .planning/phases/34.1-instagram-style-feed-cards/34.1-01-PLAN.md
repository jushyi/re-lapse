**Design direction:**

- Instagram-style: Full-width photos (edge-to-edge within card margins)
- Dark background (colors.background.primary)
- User info row BELOW photo: profile photo (small circle) + displayName + timestamp
- Reactions row below user info (same as current)
- No Polaroid frame padding - photo dominates, info is secondary
- Clean, minimal, modern aesthetic matching dark theme

**What changes from Polaroid design:**

- Remove FRAME_PADDING_TOP, FRAME_PADDING_HORIZONTAL (photo goes to card edges)
- Remove thick FRAME_PADDING_BOTTOM (info section becomes separate row below photo)
- Remove polaroid.dark background frame color - use transparent or background.primary
- Add profile photo (40x40 circle) from user.profilePhotoURL
- Info section repositioned: profile pic + name + timestamp in horizontal row
- Reactions row remains below info section

**FeedPhotoCard.styles.js changes:**

9. displayName: colors.text.primary, fontSize: 14, fontWeight: '600'
10. timestamp: colors.text.secondary, fontSize: 12
11. reactions row: paddingHorizontal: 12, paddingBottom: 12, keep existing emoji styles
